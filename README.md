# E-Commerce-Sales-Dashboard-using-Power-BI-and-SQL
This project presents an interactive E-commerce Sales Dashboard built using Power BI and PostgreSQL to analyze sales performance and generate actionable business insights.  The dashboard provides a comprehensive view of sales data across categories, regions, and time periods, helping stakeholders make data-driven decisions.

Key Features:
 1. Sales Performance Analysis across states and categories
 2. Time-Series Trends to track monthly revenue patterns
 3. Order Insights from ~98K+ transactions
 4. Revenue Analysis (~14M+ total sales)
 5. Customer Behavior Insights based on ratings and purchasing patterns
 6. Operational Metrics like On-time Delivery %

KPIs Tracked:
 1. Total Revenue
 2. Average Order Value (AOV)
 3. Year-over-Year (YoY) Growth
 4. On-time Delivery Percentage
 5. Customer Ratings
